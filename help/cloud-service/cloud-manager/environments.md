# Utilização do Adobe Cloud Manager - Ambientes

[Os ambientes do Cloud Manager](https://experienceleague.adobe.com/docs/experience-manager-cloud-manager/using/how-to-use/manage-your-environment.html) são compostos do AEM Author, do AEM Publish e do Dispatcher Services. Diferentes ambientes oferecem suporte a funções e podem ser envolvidos usando diferentes Pipelines de CI/CD. Os ambientes do Cloud Manager normalmente têm um ambiente de produção, um ambiente de preparo e um ambiente de desenvolvimento.
