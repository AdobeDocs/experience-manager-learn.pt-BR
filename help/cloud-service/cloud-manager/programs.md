# Uso do Adobe Cloud Manager - Programas

O Cloud Manager **Programas** representa conjuntos de ambientes AEM que suportam conjuntos lógicos de iniciativas de negócios, normalmente correspondendo a um Contrato de nível de serviço (SLA) adquirido. Por exemplo, um Programa pode representar os recursos AEM para suportar os sites públicos globais, enquanto outro Programa representa um DAM central interno.
