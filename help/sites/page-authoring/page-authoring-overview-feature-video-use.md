# Criação de uma página {#authoring-page-aem-sites}

Saiba como criar uma página no Adobe Experience Manager Sites usando a interface do usuário do Editor de sites.

>[!VIDEO](https://video.tv.adobe.com/v/31828?quality=12&learn=on)

Neste vídeo, os seguintes recursos são abordados:

* Criar uma página com base em um modelo
* Introduz diferentes modos de criação para uma página
   * Editar
   * Layout
   * Desenvolvedor
   * Timewarp  
* Recursos do editor de páginas
   * Localizador de componentes
   * Localizador de ativos
* Adicionar conteúdo a um componente existente
* Adicionar um novo componente à sua página
* Propriedades da página
   * Publicar página
   * Cancelar publicação da página
   * Editar modelo
   * Exibir como publicado
