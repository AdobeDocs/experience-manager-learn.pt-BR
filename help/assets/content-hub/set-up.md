# Configurar o AEM Assets Content Hub

Saiba como configurar o Experience Manager Assets Content Hub usando o programa Adobe Cloud Manager e o complemento de ativos. Saiba também como configurar o acesso para usuários que usam perfis de produto.

>[!VIDEO](https://video.tv.adobe.com/v/3472918/?learn=on)


## Configurar APIs abertas do AEM Assets Dynamic Media

Se você tiver APIs abertas do AEM Assets Dynamic Media licenciadas, poderá usá-las para gerar representações dinâmicas no Content Hub, como Recortes inteligentes. Antes que essas representações de OpenAPI do Dynamic Media possam aparecer no Content Hub, é necessária uma configuração inicial.
