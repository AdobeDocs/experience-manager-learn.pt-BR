# AEM Assets Content Hub

![AEM Assets Content Hub](./assets/overview/hero.png){align=center}

Saiba como fazer upload de ativos, aplicar tags e entender o fluxo de trabalho de aprovação no Adobe Experience Manager Assets as a Cloud Service Content Hub.


## Vídeos do Content Hub

Esta série de vídeos aborda tudo o que você precisa para trabalhar de forma eficiente com o AEM Assets Content Hub. Do gerenciamento de ativos à colaboração com sua equipe, cada vídeo fornece etapas claras e dicas práticas para ajudá-lo a aproveitar ao máximo o AEM Assets Content Hub em seu trabalho diário.
